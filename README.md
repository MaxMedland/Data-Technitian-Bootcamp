# Data Technician Skills Bootcamp | Portfolio

This repository showcases the technical projects and professional development completed during the **Leep Talent Skills Bootcamp in Data Technician**. This 7 to 12-week, government-funded intensive program is designed to master the full data lifecycle, from initial sourcing to professional stakeholder presentation.

---

## 🎓 Programme Overview
The bootcamp focuses on bridging the gap between raw data and actionable business insights. It combines technical "hard" skills in data manipulation with "soft" transferable skills required for a career as a Data Technician or Data Analyst.

### 🧠 Knowledge Gained
* **The Data Lifecycle:** Understanding how to source, gather, cleanse, validate, and analyze data to identify patterns.
* **Data Governance:** Essential knowledge of GDPR, data security, and ethical considerations in data handling.
* **Database Fundamentals:** Learning relational database structures and how to organize data for efficiency.
* **Cloud Analytics:** Understanding how modern organizations leverage cloud environments for data storage and processing.

---

## 🛠️ Software & Technical Stack
Throughout the bootcamp, I gained proficiency in industry-leading tools:
* **Microsoft Excel:** Advanced formulas, PivotTables, and data modeling.
* **SQL:** Querying relational databases to extract specific information.
* **Python:** Data manipulation using **Pandas** and **NumPy**; visualization with **Matplotlib** and **Seaborn**.
* **Power BI & Tableau:** Building interactive business intelligence dashboards.
* **Microsoft Azure:** Exploring cloud data fundamentals and analytics workloads.
* **GitHub:** Version control and professional portfolio management.

## 📊 Data Skills Acquired
* **Data Cleansing & Transformation:** Converting "dirty" data into structured formats using Excel and Python.
* **Advanced Statistical Analysis:** Using functions like `SUMIFS`, `AVERAGEIF`, and `VLOOKUP` to derive summary statistics.
* **Logical Categorization:** Implementing complex logic (e.g., the `SWITCH` function in Excel) to automate data labeling.
* **Data Storytelling:** Translating complex datasets into visual charts, graphs, and PivotCharts that highlight trends and market opportunities.

---

## 🤝 Transferable Professional Skills
The program emphasizes workplace readiness through:
* **Analytical Problem Solving:** Turning vague business questions into data-driven solutions.
* **Technical Communication:** Drafting reports and "elevator pitches" to explain data findings to non-technical audiences.
* **Project Management:** Working to **SMART goals** and managing tasks within a group project environment.
* **Personal Branding:** Optimizing professional presence via LinkedIn and building a technical portfolio on GitHub.
